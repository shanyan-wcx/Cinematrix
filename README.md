# Cinematrix

毕业设计 - 影视资源搜索整理系统

------

## 开始使用

首先，克隆该项目：

```bash
git clone https://github.com/shanyan-wcx/Cinematrix.git
```

接着，进入项目文件夹：

```bash
cd Cinematrix
```

然后，安装所需依赖：

```bash
npm install
```

最后，启动程序：

```bash
npm start
```

现在，服务已经运行在[http://localhost:3000](http://localhost:3000)，请在浏览器中打开该地址。
